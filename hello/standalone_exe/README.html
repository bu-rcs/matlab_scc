RCS Matlab Example Codes

    This directory contains an exmaple to make a Matlab standalone executable and to submit a job to execute the executable.

  • hello.m: A MATLAB code to print "Hello World".

    • To compile it to obtain a standalone executable,

    mcc -mv -o hello hello.m

    To submit a batch job to execute the standalone executable,

    qsub job_standalone.qsub

  • job_standalone.qsub: a script to submit a job using Matlab standalone executable.
