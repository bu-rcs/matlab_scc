RCS Matlab Example Codes

    This directory includes a hello world program and a sample script to submit MATLAB jobs on SCC.

  • hello.m: A MATLAB code to print "Hello World". To submit a batch job to run it on an SCC compute node,

    • qsub job_mfile.qsub

  • job_mfile.qsub: a script to submit a job using Matlab m-file.

  • standalone_exe: This directory contains an exmaple to make a Matlab standalone executable and to submit a job to execute the executable.
