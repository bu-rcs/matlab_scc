RCS Matlab Example Codes

    This directory includes a hello world program and sample scripts to submit MATLAB jobs on SCC.

  • hello.m: A MATLAB code to print "Hello World". To submit a batch job to run it on an SCC compute node,

    • qsub job.mfile

    To compile it to obtain a standalone executible,

    mcc -mv -o hello hello.m

    To submit a batch job to run the standalone executible,

    qsub job.standalone

  • job.mfile: a script to submit a job using Matlab m-file.

  • job.standalone: a script to submit a job using Matlabe standalone executable.

  • job.array: a script to submit job array
